# 🚀 ErrorLens

**ErrorLens** is an AI-powered debugging-focused IDE designed to help developers understand, trace, and fix errors faster.

> Not just a code editor — a debugging workspace.

---

## ✨ Features (Current)

* ⚡ Electron-based desktop app
* 🧠 Monaco Editor integration
* 📁 Open folder & explore files
* 📄 Click-to-open file system
* 🖥️ Cross-platform (Windows + macOS)

---

## 🔥 Vision

ErrorLens is built to go beyond traditional editors like VS Code or Cursor.

### Goal:

* Detect errors intelligently
* Explain issues clearly
* Suggest fixes instantly
* Provide AI-assisted debugging

---

## 🏗️ Tech Stack

* Electron
* React + TypeScript
* Monaco Editor
* Node.js

## 🚀 Getting Started

### 1. Clone repo

```
git clone https://github.com/vinit105/error_lens_code.git
cd error_lens_code/apps/desktop
```

### 2. Install dependencies

```
npm install
```

### 3. Run app

```
npm run dev
```
